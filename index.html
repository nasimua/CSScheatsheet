CSS Cheatsheet

Selector Types

The following list of selectors becomes more specific as you make your way down the list.

  1. Type Selectors - e.g. h1, h2, etc.
  2. Class Selectors - e.g., .class (defined in html file in following format; class="class")
  3. ID Selectors - e.g., #id (defined in html file in following format; id="id")

Common CSS properties

The following table describes a few of the CSS properties that you can use to make your webpage stand out!

Property Description
color Sets the color of a text
line-height Sets the distance between lines
letter-spacing Sets the distance between characters
text-align Aligns the text in an element
border Sets border properties in one declaration (width, style and color)
font-family Sets the font family of the text, e.g. Helvetica
font-size Sets the size of the text, e.g. small, large, 10px, 0.4em
font-weight Sets the weight of text, e.g. bold, normal, 100-900, etc.
